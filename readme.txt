Customise WordPress with powerful, professional and intuitive post types

== Description ==

Advanced Custom Post Types is the perfect solution for any WordPress website which needs more content types like other Content Management Systems.

== Installation ==

1. Upload 'advanced-custom-post-types' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
